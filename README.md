# 🚀 Resume Flow – Free Online Resume Builder

Resume Flow is a completely free and easy-to-use online resume builder that helps users create professional resumes in minutes. Built with modern web technologies like **Vite, React, JavaScript, and Tailwind CSS**, it offers a fast, responsive, and smooth user experience.

---

## 🌐 Live Demo

👉 https://resume-flowz.vercel.app

---

## 📌 Features

- ✨ Clean and modern UI
- ⚡ Fast performance using Vite
- 📱 Fully responsive design (mobile-friendly)
- 🧾 Easy resume creation with structured sections
- 🎨 Styled using Tailwind CSS
- 🔄 Real-time preview
- 💾 Completely free to use

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS v4.2  
- **Language:** JavaScript  

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/adnan-bhaldar/Resume-Flow.git

# Navigate into the project
cd Resume-Flow

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🚀 Build for Production

```bash
npm run build
```
